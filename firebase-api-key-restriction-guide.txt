Firebase API 키 보호 안내

정적 GitHub Pages 앱에서는 Firebase 웹 API 키가 코드에 보일 수 있습니다.
상용/공개 배포 시 최소한 아래 설정을 반드시 적용하세요.

1. Google Cloud Console 접속
2. API 및 서비스 > 사용자 인증 정보
3. 현재 웹 API 키 선택
4. 애플리케이션 제한사항: HTTP 리퍼러(웹사이트)
5. 허용 도메인 예시
   - https://99ksy9990-bot.github.io/*
   - https://99ksy9990-bot.github.io/family-cal/*
6. API 제한사항
   - Firebase 관련 API만 허용
7. 저장 후 앱 동작 확인

주의:
- API 키를 index.html 밖으로 옮겨도 GitHub Pages에 올리면 결국 공개될 수 있습니다.
- 진짜 권한 보호는 Firestore Security Rules와 Firebase Authentication으로 처리해야 합니다.
